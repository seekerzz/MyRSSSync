| [Spoken DialogSum: An Emotion-Rich Conversational Dataset for Spoken Dialogue Summarization](https://arxiv.org/abs/2512.14687) | ### 贡献点：<br/><br/>1. **数据集Spoken DialogSum的构建**：论文引入了Spoken DialogSum，这是首个将原始对话音频、事实性摘要、情感丰富的摘要以及说话者年龄、性别和情绪等单句级标签相结合的数据集。该数据集通过两个阶段构建：首先，利用LLM对DialogSum脚本进行重写，并添加Switchboard风格的填充物与反馈链；然后，为每一句话标记情感、音高及话语节奏。<br/><br/>2. **TTS引擎的应用**：论文使用了具有表现力的TTS（文本转语音）引擎，从带有标签的脚本中合成音频，并与旁白标签进行对齐，以生成对话音频样本。<br/><br/>3. **数据集规模**：Spoken DialogSum包含13,460个情感多样化的对话，每个对话都配有一个事实性的摘要和一个关注情绪的摘要。这为研究提供了丰富的资源，旨在探索更复杂的情感与语境关系在语音交互中的影响。<br/><br/>4. **在线演示**：论文提供了一个在线演示（https://fatfat-emosum.github.io/EmoDialog-Sum-Audio-Samples/），供公众访问Spoken DialogSum的音频样本，同时计划在未来不久发布完整数据集。<br/><br/>5. **模型评估与基准测试**：通过比较基于语音识别（ASR）和语言模型（LLM）串联系统的性能与基于端到端语音建模的Audio-LLM系统在情感摘要任务上的表现，论文验证了全链条语音模型的价值。结果显示，相比于分段式方法，端到端的语音建模策略显著提高了情感摘要的相关指标ROUGE-L，增加了28%。<br/><br/>这些贡献共同推动了情绪感知或口语对话摘要领域的发展，并为相关研究提供了一个宝贵的数据集和评估基准。 |
