| [JASTIN: Aligning LLMs for Zero-Shot Audio and Speech Evaluation via Natural Language Instructions](https://arxiv.org/abs/2605.04505) | 贡献点如下：<br/><br/>1. **提出了一种新的评估框架** - JASTIN（Generalizable Audio and Speech Task Inference Network）是一个面向音频和语音任务的通用评估体系，用于衡量生成式音频模型的能力。<br/><br/>2. **解决现有问题** - 该论文针对当前评估方法与生成式音频模型快速发展之间的不匹配进行了探讨，并指出现有的客观指标和一般多模态大型语言模型（MLLMs）在领域泛化、零样本能力和指令灵活度方面存在挑战。<br/><br/>3. **自指示的音频评估框架** - JASTIN将音频评估设计为一个自我指导推理任务，通过引入可训练的音频适配器，结合冷冻高性能音频编码器和微调后的LLM基础结构来实现这一目标。这使得模型能够在不特定任务重训练的情况下，在语音、声音、音乐以及跨域评估任务中表现出色。<br/><br/>4. **全面的指令遵循数据准备管道** - 为了确保JASTIN在零样本泛化上的表现，论文引入了一个综合性的数据准备流程，其中包括多源、多任务、多校准和多描述等方法来增强模型的适应性和通用性。<br/><br/>5. **实验结果** - 实验结果显示，JASTIN在与人类主观评分的相关度上达到了行业领先水平（通过Pearson和Spearman相关系数评估），并且其性能显著优于一般的MLLMs。这一优势跨越了语音、声音、音乐等多个领域，并且无需针对特定任务进行重新训练。<br/><br/>6. **创新的评估方法** - JASTIN提供了一种新颖的方法来评估生成式音频模型，通过结合高性能的音频编码器和可调的LLM架构，以及独特的数据准备策略，能够更有效地评估模型在不同场景下的表现。 |
| [Spatial-Magnifier: Spatial upsampling for multichannel speech enhancement](https://arxiv.org/abs/2605.04749) | ### 贡献点：<br/><br/>1. **提出Spatial-Magnifier** - 一种设计用于从有限的真实麦克风测量数据生成虚拟麦克风（VM）信号的神经网络。这一创新解决了将大型捕捉阵列集成到实际边缘设备时由物理限制所导致的空间直接性局限问题。<br/><br/>2. **引入SARL框架（Spatial Audio Representation Learning Framework）** - 该框架利用估计出的虚拟麦克风信号和特征来条件化下游语音增强系统，以提升音频处理能力。这一方法能够通过有限的真实麦克风数据预测和改善全阵列时的性能。<br/><br/>3. **实验验证优越性** - 实验结果表明，所提出的框架在各种语音提取系统中（包括端到端多通道语音增强和神经束形成）都超越了现有的空间上采样基准线。证明该方法几乎能够达到所有麦克风可用时的最佳表现。<br/><br/>4. **克服物理限制** - 通过使用虚拟麦克风信号和SARL框架，Spatial-Magnifier提供了一种在物理空间资源有限的情况下提升音频处理效果的新途径，特别适用于实际边缘设备的应用场景。<br/><br/>5. **多模态应用潜力** - 借助该方法和框架，未来可以在包括但不限于机器人、智能家居设备等需要高精度语音识别与增强的领域中，提高语音处理系统的性能。 |
