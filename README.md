| [Multifaceted Evaluation of Audio-Visual Capability for MLLMs: Effectiveness, Efficiency, Generalizability and Robustness](https://arxiv.org/abs/2504.16936) | 贡献点如下：<br/><br/>1. **全面评估模型的音频视觉能力**：论文提出了一种对多模态大型语言模型（MLLMs）在处理多种模态信息时的音频视觉能力进行多方面评估的方法。这一评估覆盖了多种场景，包括分布变化和对抗攻击等。<br/><br/>2. **四个关键维度**：评估从四个维度展开，即有效性（effectiveness）、效率（efficiency）、泛化性（generalizability）和鲁棒性（robustness），全面考察MLLMs在处理音频视觉信息时的表现与限制。<br/><br/>3. **零样本与少量样本下的强大能力**：实验表明，MLLMs在数据有限的情况下仍然能展现出强大的零样本学习能力和少量样本学习能力，意味着它们能在缺乏大量训练数据的支持下达到优异的性能。<br/><br/>4. **对视觉模态的高度依赖**：研究发现，尽管MLLMs表现出色，但其表现很大程度上依赖于视觉输入。当视觉信息受到干扰或缺失时，模型的整体性能会受到影响。<br/><br/>5. **对抗样本下的相对鲁棒性**：与传统模型相比，MLLMs在面对对抗样例（adversarial samples）时展现出较高的鲁棒性，这表明它们能够更有效地抵御特定类型的攻击和挑战。 |
